pub fn random(&self, subject: &[u8], output: &mut &mut [u8])

Returns a randomized hash.

Note
  • This is the off-chain environment implementation of random. It provides the same behavior in that it will likely yield the same hash for the same subjects within the same block (or execution context).
Example
   let engine = ink_engine::ext::Engine::default();
   let subject = [0u8; 32];
   let mut output = [0u8; 32];
   engine.random(&subject, &mut output.as_mut_slice());
