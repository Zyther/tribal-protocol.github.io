pub fn pull_spread_root<T>(root_key: &Key) -> T where
    T: SpreadLayout, 
Expand description

Pulls an instance of type T from the contract storage using spread layout.

The root key denotes the offset into the contract storage where the instance of type T is being pulled from.

Note

  • The routine assumes that the instance has previously been stored to the contract storage using spread layout.
  • Users should prefer using this function directly instead of using the trait methods on SpreadLayout.
